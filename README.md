## 🖼️ 기획 배경
### 스터디 + MBTI = "STU-TI"
개발자라면 한두개는 기본으로 참여하고 있다는 스터디! 
<blockquote> "나와 더 잘 맞는 팀원을 만나 오래 유지할 수 없을까?" </blockquote>

개발자들이 <b>본인과 궁합이 좋은 MBTI를 기준으로 스터디를 만들고, </b> <br/>
배운 지식을 <b>커뮤니티를 통해 함께 공유</b>하는 애플리케이션입니다.
